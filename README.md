# Python Learning Record

このリポジトリは、Pythonの基礎構文から実践的なコーディングスキル、およびデータ処理の基礎を身につけるための学習記録です。
質の高い公開教材を活用し、自学自習で進めた成果としてのコードをまとめています。

## 📚 学習教材と内容

このリポジトリには、主に以下の2つの学習記録が含まれています。

### 1. 東京大学 「Pythonプログラミング入門」
- **概要**: 東京大学が公開している教材を用いた学習記録です。変数の扱い、制御構文、基本的なデータ構造など、Pythonの土台となる知識を体系的に学んでいます。
- **出典**: [Pythonプログラミング入門 — Pythonプログラミング入門 documentation](https://utokyo-ipp.github.io/)
- **ディレクトリ**: `/utokyo_python` (※ご自身のフォルダ名に合わせて変更してください)

### 2. Python 入門100本ノック
- **概要**: 基礎的なアルゴリズムやデータ操作、文字列処理などのロジックを組む練習として取り組んだコードです。
- **出典**: [Python 入門100本ノック #初心者向け - Qiita](記事のURLをここに貼る)
- **ディレクトリ**: `/100_knocks` (※ご自身のフォルダ名に合わせて変更してください)

## 🛠 開発環境
- Python 3.x
- Google Colaboratory (Jupyter Notebook)

## 📝 ノートと工夫した点
- 学んだことや、つまづいたエラーとその解決策は、コード内のコメント（または各ファイル）にメモとして残しています。
- 単に正解を書き写すのではなく、なぜその挙動になるのか（例：エスケープシーケンスの仕様など）を検証しながら進めています。
